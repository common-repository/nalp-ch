This Plugin allows easy integration of nalp.ch appointment booking into your wordpress website.

== Description ==

Just install the plugin and add the Nalp.ch Button to your page. Do not forget to configure the slug. You can find the slug on https://admin.nalp.ch

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/nalp` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add a new block "nalp" from the Widgets section
